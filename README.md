# sudoku_solver_cpp
Sudoku solver in C++

# Run with g++

```
g++ -std=c++17 solver.cpp
./a.out
```

Input: sudoku board with 0's for empty spots and spaces between elements:
```
0 0 8 0 0 0 0 0 0
4 9 0 1 5 7 0 0 2
0 0 3 0 0 4 1 9 0
1 8 5 0 6 0 0 2 0
0 0 0 0 2 0 0 6 0
9 6 0 4 0 5 3 0 0
0 3 0 0 7 2 0 0 4
0 4 9 0 3 0 0 5 7
8 2 7 0 0 9 0 1 3
```

Output: animated backtracking to solve
